CSS Color Module

Le CSS color module est une façon de décrire une couleur afin de l'appliqué sur un élément HTML.

Les couleurs peuvent être appliquées dans plusieurs contexte, afin de choisir la couleur d'un fond, d'une police d'écriture ou même d'effets plus complexes tel que des ombres. Les propriétés "background-color" et "color" s'appuyent donc sur le color module CSS.

Exemple pour définir le fond de cette carte

La carte ci-dessus possède l'attribut de style "background-color: pink". Pink est donc compris comme une couleur est peut être utilisé pour définir une couleur.

Le color module accepte un mot, un code hexadécimal, un code de 3 octets (1 pour chaque couleur primaire. Rouge, Bleu, Vert).

Le format pour définir une couleur en faisant le mélange des 3 couleurs primaires est le suivant :

background-color:
rgb(0, 255, 0);

La couleur ci-dessus est vert, on pourrait aussi l'écrire :

background-color:
rgb(0, 100%, 0);

L'équivalent en utilisant le nom d'une couleur :

background-color:
lime;

Il existe aussi la fonction hsl(nuance, saturation, luminosité). La nuance est défini en degrès représentant l'angle sur un cercle chromatique, 0deg étant le rouge, 120deg le vert et 240 le bleu. 360deg et 720deg auront pour même résultat que 0deg (rouge).

background-color:
hsl(120deg 100% 50%);

Dans la fonction utilisé, la saturation est à 100%, réduire cette valeur ferais tiré la couleur vers le gris. Le paramètre de la luminosité à 50% donne la couleur normal, à 100% on aurait du blanc, et du noir à 0%.

Sans oublié l'hexadécimale :

background-color:
#00ff00;

Les deux méthodes rgb() et hsl() sont aussi possible avec rgba() et hsla(), a pour alpha ce qui va permetre d'ajouter un pourcentage de transparence :

background-color:
rgba(0, 255, 0, 0.5);
background-color:
rgba(0, 255, 0, 0.5);

L'héxadécimale peut prendre 2 octets supplémentaires pour géré la transparence :

background-color:
#00FF007F;

Le mot "transparent" peut être utilisé aussi comme nom mais défini quelque chose de transparent. Le fait que il ne défini pas une opacité, celui ci est plus destiné à être utilisé pour les dégradés.
